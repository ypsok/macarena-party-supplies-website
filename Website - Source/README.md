# Macarena Party Supplies Website

Sitio web estatico en HTML, CSS y JavaScript vanilla para un negocio de diseno impreso y digital.

## Objetivo

Crear una experiencia clara para explorar servicios, iniciar una cotizacion y enviar un resumen de pedido por WhatsApp. El sitio esta pensado para publicarse en GitHub Pages y crecer por etapas sin React ni dependencias de build.

## Flujo actual

1. `index.html`: pagina principal, identidad del negocio, categorias, colecciones, proceso y contacto.
2. `services.html`: servicios comerciales por tipo de cliente/uso.
3. `new-order.html`: cotizador de pedido con catalogo estatico, personalizacion, resumen y enlace `wa.me`.

## Siguientes etapas recomendadas

1. Separar el catalogo a un archivo `data/products.json` o `js/data/products.js`.
2. Crear una pagina de catalogo con filtros y tarjetas de producto.
3. Agregar galeria local de referencias visuales inspirada en Instagram.
4. Preparar una API externa para guardar pedidos en Neon/Postgres. GitHub Pages no debe conectarse directo a la base de datos.
5. Definir paginas legales basicas: privacidad, terminos y politica de pedidos personalizados.

## Desarrollo local

Puedes abrir el proyecto con VSCode y usar Live Server o FiveServer desde esta carpeta.
